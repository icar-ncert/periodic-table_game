📋 Game Instructions & Rules

The Periodic Table Challenge is an interactive educational tool designed to enhance understanding of chemical elements, their properties, and their positions in the periodic table.

  • Teacher Mode: Host a live classroom session. Generate a room code and invite students to compete in real-time.
  • Student Mode: Join a teacher's room using a unique Room Code.
  • Individual Mode: Practice solo to improve your speed and accuracy.

🟢 Easy Mode

Objective: Identify elements by their name.

Instructions: An element name (e.g., "Oxygen") will appear on screen. You must click the correct element box on the periodic table.

🟡 Medium Mode

Objective: Place elements in their correct positions.

Instructions: You are given 10 draggable element boxes. Drag each element to its correct position on the blank periodic table grid. This tests your knowledge of atomic number trends and group/family locations.

🔴 Hard Mode

Objective: Identify elements by specific Physical and Chemical properties.

Instructions: A property (e.g., "Atomic Mass = 15.999 u" or "Electron Configuration") is displayed. You must identify which element possesses that property.

Action Points
Correct Answer +100 Base Points
Time Bonus +0 to +200 Points
Wrong Answer -50 Points

*Time Bonus decreases by 2 points every 0.1 seconds. Answer quickly to maximize your score!
